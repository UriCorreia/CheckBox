# CheckBox Task Manager

Projeto desenvolvido com o objetivo de implementar um gerenciador simples de tarefas utilizando Java e Maven.

## Descrição

O sistema permite realizar operações básicas de gerenciamento de tarefas, como:

- Criar novas tarefas
- Listar tarefas cadastradas
- Remover tarefas

O projeto segue uma arquitetura organizada em camadas, separando responsabilidades entre lógica de negócio e persistência de dados.

---

## Tecnologias Utilizadas

- Java
- Maven
- JDBC
- Banco de dados relacional
- IntelliJ IDEA

---

## Como Executar o Projeto

### 1️⃣ Clonar o repositório

`git clone https://github.com/UriCorreia/CheckBox.git`

### 2️⃣ Compilar o projeto

`mvn clean package`

### 3️⃣ Executar

Após a compilação, o arquivo .jar será gerado na pasta:

target/

Execute com:

`java -jar nome-do-arquivo.jar`
